1. calc动态计算属性
    calc() 函数用于动态计算长度值。主要是使用于页面内动态给dom元素赋值宽高等操作
    width: 50%;
    height: 400px;
    background: yellow;
    width: calc(50% - 200px);
    height: 400px;
    background: blue;
    color: #ffff;
  2. currentcolor继承色
    currentColor代表了当前元素被应用上的color颜色值。 使用它可以将当前这个颜色值应用到其他属性上，或者嵌套元素的其他属性上。
    多用于按钮hover效果实现 简化css代码
    试试下面hover效果吧
    {
    color: blue;
    border: 1px solid currentcolor;
    box-shadow: 0 0 5px currentcolor;
    }
    :hover{
    color: green;
    }
    {
    color: red;
    border: 1px solid currentcolor;
    box-shadow: 0 0 5px currentcolor;
    }
    :hover{
    color: yellow;
    }
  3. background-clip背景、文字剪裁
    background-clip属性的通俗作用就是指定元素背景所在的区域,有四种取值
    background-clip: border-box|padding-box|content-box|text
    多用于css动画波浪效果,文字渐变色背景图实现
    background-clip：text文字剪裁
  4. object-fit
